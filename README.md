# Қонақ үй брондау жүйесі

Интерактивті веб-қосымша (Node.js + JavaScript)

## Функционалдық мүмкіндіктер

- Жаңа брондау қосу
- Брондаулар тізімін көру
- Брондауды болдырмау
- Статистиканы көру
- Деректерді JSON файлда сақтау
- EventEmitter оқиғаларын логирлеу
- Қателерді өңдеу
